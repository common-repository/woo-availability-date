Show availibilty date for woocommerce products.

== Description ==
Add product availability date to woocommerce products.
Just like Amazon.in this plugin displays the date E.G: "Buy Before Wednesday 12".
It displays on both shop and product pages.
You can assign the date and text to each product page.
Ones the product is expired it saves to 'Draft' and the product no longer displays on the website.

If any issues regarding this plugin mail me @: sandeep.tete2013@gmail.com


== Installation ==
- Upload zip file.
- Activate the plugin.
- Scroll down to 'Woocommerce Availability Date'.
- Modify the settings.
- Done.
